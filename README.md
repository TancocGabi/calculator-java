# Calculator Java

## Descriere
Aplicație de tip calculator realizată în Java, capabilă să efectueze operații aritmetice de bază.

## Funcționalități
- Adunare
- Scădere
- Înmulțire
- Împărțire

## Tehnologii utilizate
- Java
- Eclipse IDE

## Rulare
Proiectul se rulează din clasa `App.java`.
